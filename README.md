# Reader's Block v.01
A browser extension (chromium) that indulges your reading habit 5 minutes at a time!

You can also practice your typing skills by choosing a book title with the "Read as You Type" button! (if you run into special characters, and don't have a numpad / don't want to look up the numpad code, you can press ENTER twice (or is it three?) times to skip over them)

DOWNLOAD ZIP - UNPACK ZIP - TURN ON DEVELOPER MODE IN YOUR BROWSER EXTENSIONS OVERVIEW - LOAD FOLDER AS UNPACKED EXTENSION - GITHUB PACKS MAIN FOLDER AS SUB FOLDER TO FOLDER NAMED  "Reader-s-Block-main"
